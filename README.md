# FraudGuard-AI
FraudGuard AI — ML fraud detection on 284K imbalanced transactions (0.17% fraud rate). XGBoost classifier with threshold tuning and cost-sensitive weighting achieves 88% fraud recall and PR-AUC 0.81. Covers imbalanced classification, feature importance analysis, and precision-recall tradeoffs. Python · Scikit-learn · XGBoost
